# 指板记忆 🎸

一个帮助你熟悉吉他指板的练习工具。通过游戏化的方式，让你在不知不觉中记住每个位置的音符。

## 功能特性

### 音符定位模式
- 系统随机出一个音符（如 A3、C4）
- 你需要找到指板上对应的位置并弹奏
- 麦克风实时检测你弹奏的音符
- 即时反馈正确/错误

### 难度分级
- **初级**：0-5 品，适合新手
- **中级**：0-9 品，进阶练习
- **高级**：全指板，挑战自我

### 练习统计
- 正确率统计
- 平均反应时间
- 连续答对计数
- 历史记录

## 技术栈

- **前端框架**：React 19 + TypeScript
- **构建工具**：Vite 8
- **样式**：Tailwind CSS 4
- **状态管理**：Zustand
- **音频处理**：Web Audio API（自相关算法）
- **PWA**：支持离线使用、添加到桌面

## 快速开始

### 安装依赖
```bash
cd guitar-practice
npm install
```

### 开发模式
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

### 预览生产版本
```bash
npm run preview
```

## 使用说明

1. 点击"开始练习"
2. 允许麦克风权限
3. 系统会随机出一个音符
4. 在吉他上找到该音符并弹奏
5. 系统会检测你弹奏的音符并给出反馈

## 设计理念

- **极简**：界面干净，专注于练习本身
- **温柔**：暖色调配色，舒适的视觉体验
- **慢**：不追求速度，享受熟悉指板的过程
